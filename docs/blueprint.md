# **App Name**: MJL Transport Hub

## Core Features:

- Landing Page with Flier: Display the MJL Transport flier image from 'https://crwwlgwjdclhvwqwspoa.supabase.co/storage/v1/object/public/mjlt/Minimalist%20Square%20Photo%20Instagram%20Post.png' on the landing page.
- Book a Ride CTA: Implement a 'Book a Ride' call-to-action button that navigates users to the booking form.
- Booking Form: Create a form to collect user details: Full Name, Location (Lagos, PH City, Abuja), Address, Hostel (Male Hall 1-7, Med Male, Wema, ABUAD, NFH 1 & 2, Female Med), Number of Luggage's (with extra fee disclaimer), Preferred Vehicle (8-seater/5-seater), Departure Date.
- WhatsApp Submission: After form submission, direct collected data to 'https://wa.me/2349135067590' via WhatsApp.

## Style Guidelines:

- Primary color: Deep purple (#5E17EB) for a modern and trustworthy feel.
- Background color: Light gray (#F2F2F2) to provide a clean and professional backdrop.
- Accent color: Bright red (#FF0000) to draw attention to the 'Book a Ride' button and other important elements.
- Font: 'Glacial Indifference', a sans-serif, as requested by the user. Note: currently only Google Fonts are supported.
- Single-page layout with a clear visual hierarchy.
- Simple and clear icons for form fields where applicable to improve usability.
